# 📈 API de Cotação de Moedas

Projeto desenvolvido em Python com o objetivo de consumir uma API externa de cotações e disponibilizar informações utilizando **FastAPI**.

A aplicação consulta a **AwesomeAPI**, obtendo dados atuais de moedas e Bitcoin, além de realizar uma análise histórica do Bitcoin com geração de gráfico utilizando Matplotlib.

---

## 🚀 Funcionalidades

- Consulta de cotação atual:
  - 💵 Dólar (USD/BRL)
  - 💶 Euro (EUR/BRL)
  - ₿ Bitcoin (BTC/BRL)

- Consulta do histórico do Bitcoin:
  - Janeiro a Junho de 2026
  - Cálculo da média mensal
  - Visualização gráfica da evolução dos preços

- API própria utilizando FastAPI:
  - Endpoint inicial da aplicação
  - Endpoint para consulta das cotações

---

## 🛠️ Tecnologias utilizadas

- Python 3
- FastAPI
- Requests
- Matplotlib
- AwesomeAPI
